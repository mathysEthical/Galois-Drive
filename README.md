# Galois Drive

Galois Drive is a web application enabling you to use Google Drive to store files securely using AES-GCM.

![Galois Drive](./public/img/GaloisDrive.png)

## Implemented
- [x] Login with Google
- [x] Upload files
- [x] Download files
- [x] Delete files
- [x] Right click menu
- [x] Client side encryption
- [x] Rename files
